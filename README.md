# cybernagle/homebrew-tap

Homebrew tap for CyberNagle projects.

## Usage

```bash
brew tap cybernagle/tap
```

## Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [memory-cli](https://github.com/cybernagle/memory-cli) | Unified memory management for multiple AI agents | `brew install memory-cli` |
| [fingersaver](https://github.com/cybernagle/fingersaver) | AI coding agent orchestrator with split-pane TUI | `brew install fingersaver` |
